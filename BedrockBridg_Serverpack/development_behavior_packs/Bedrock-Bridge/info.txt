BedrockBridge v1.6.7 by EsploratoriDevelopment
compatible with Bedrock 1.21.90 (till the next major update)

INSTALLING
1) Invite the bot to your server from this link https://discord.com/api/oauth2/authorize?client_id=1041838898843762769&permissions=2684357632&scope=bot%20applications.commands
2) Run /download with the desired options
3) Install the pack to your world
4) Enable beta-api (scripting) in the world
5) Replace the file `<main bds folder>/config/default/permissions.json` with the following:
https://github.com/InnateAlpaca/BedrockBridge/blob/main/BedrockBridg_Serverpack/config/default/permissions.json

A more detailed guide can be found at https://github.com/InnateAlpaca/BedrockBridge
